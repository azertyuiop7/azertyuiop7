K.exe

Pour l'installer : K-installer ou entrez dans l'invite de commande :

C:\>powershell -Command "Invoke-WebRequest https://1089x.github.io/K.exe -Outfile K.exe"

Utilisation : Entrez K fonction() dans une invite. Par exemple K t(60) sourire(15) pour afficher un sourire pendant 15 secondes une minute après le lancement de la commande.

Liste des fonctions :

❖ background(image) : Met en fond d'écran l'image de chemin d'accès ou d'url "image"
❖ background_color(r,g,b) : Met la couleur rbg(r,g,b) en fond d'écran
❖ background_rainbow() : Change la couleur du fond d'écran en dégradé
❖ bubbles(duration) : Affiche des bubulles sur l'écran. (écran de veille)
❖ cmd_d(n) : Effectue cmd+d 10*n fois
❖ download_file(url) : Télécharge le fichier d'url 'url' dans le répertoire d'exécution
❖ hacke(temps) : Ouvre une fenêtre en plein-écran avec un message «hacké», et change de couleur toutes les 20ms pendant environ 'temps' secondes
❖ kill_explorer(duration) : Kill le processus explorer.exe pendant duration secondes
❖ kill(processus) : Tue l'application de nom "processus"
❖ piano() : Joue une note lorsqu'une touche est frappée
❖ press(keys) : Appuie les touches "keys". Touches spéciales : *W → Cmd, *C → Ctrl, *A → Alt, *S → Shift. Par exemple : "*Cc" → Ctrl + c
❖ replace_letters() : Remplace les lettres frappées
❖ sourire(n,color) : Affiche un sourire à l'écran pendant environ n*1.5 secondes, de couleur hexadécimale "BBGGRR."
❖ start(processus) : Ouvre l'application de nom "processus"
❖ cmd(commande) : effectue la commande batch "commande"
❖ t(min:sec) : Délai d'attente de "min:sec" avant d'effectuer les commandes suivantes. Les messages d'erreur ne sont pas affichés si cette fonction est utilisée
❖ update : Télécharge la dernière version de K
